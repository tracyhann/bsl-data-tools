# Data Resource Tools

Small Colab-based utilities for preparing, cleaning, and navigating study data resources.

| Tool | What it does |
| --- | --- |
| [REDCap Cleaner](redcap-cleaner/README.md) | Cleans REDCap exports into traceable, analysis-ready Excel workbooks. |
| [Folder Navigator](folder-navigator/README.md) | Turns a Google Drive folder link into a clickable directory tree. |
| [Study Folder Initializer](study-folder-initializer/README.md) | Creates a new study Google Drive folder from a reusable template. |

Each tool README includes its Colab link, quick start, and usage tips.
